# WebVisJson

将 json 数据可视化



## 软件框架

Vue

## 快速开始

## Project setup

```
npm install --registry=https://registry.npmmirror.com
```

### Compiles and hot-reloads for development

```
npm run dev
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```

### 提交 build 后的网页

```
npm run deploy
```
